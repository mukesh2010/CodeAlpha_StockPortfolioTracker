# CodeAlpha_StockPortfolioTracker

## Description
A simple console-based Stock Portfolio Tracker built in Python as part of the CodeAlpha Python Programming Internship (Month 1, Task 2). The program calculates the total investment value based on user-entered stock symbols and share quantities, using a hardcoded dictionary of stock prices.

## Features
- Predefined dictionary of stock prices (e.g., AAPL, TSLA, GOOGL, AMZN, MSFT)
- Accepts multiple stock entries from the user until they type "done"
- Calculates individual and total investment value
- Optional feature to save the portfolio summary to a CSV file

## Technologies Used
- Python 3
- Concepts: dictionary, input/output, basic arithmetic, file handling

## How to Run
1. Clone this repository
2. Run `python stock_tracker.py` in your terminal
3. Enter stock symbols and share counts when prompted
4. Type `done` when finished to see the total investment value
5. Choose whether to save the results to a CSV file
